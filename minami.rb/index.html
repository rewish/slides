jQueryの基本的な文法

特に理由が無ければ以下のように書く。

<script>
  jQuery(document).ready(function($) {
    // なんか処理
  });

  もしくは

  jQuery(function($) {
    // なんか処理
  });
</script>

jQueryの基本的な文法

ちなみにコレはダメな例

$(document).ready(function() {
  // なんか処理
});

$(function() {
  // なんか処理
});

特徴1: CSSセレクタで要素を取得

jQuery(function($) {
  $('div p').css({backgroundColor: '#F00'});

DOMを操作するメソッド

先ほどcss()とtext()を使いましたが、他にも様々なメソッドが用意されています。

  • css()
  • text()
  • attr()
  • removeAttr()
  • addClass()
  • removeClass()
  • toggleClass()
  • html()
  • val()

DOMを操作するメソッド：軽く説明

// Getter
$('div p').attr('title');

// Setter
$('div p').attr('title', 'たいとる');

// Getter
$('div p').css('backgroundColor');

// Setter
$('div p').css('backbroundColor', '#FFF');

DOMを操作するメソッド：軽く説明

// Getter
$('div p').text();

// Setter
$('div p').text('てきすと');

html()やval()も同じように取ったり設定したりできる

DOMを操作するメソッド：軽く説明

$('div p').addClass('新しいclass属性');

$('div p').removeClass('消すclass属性');

$('div p').toggleClass('無ければ追加するclass属性');

イベント

jQueryならDOMイベントも簡単にバインドできます。

jQuery(function($) {
  $('div p').each(function(i) {
    $(this).click(function() {
      $(this).text(i % 2 === 0 ? 'オレ偶数！' : 'ワタシ奇数！');
    });
  });
});

エフェクト：種類

jQueryには標準で様々なエフェクトがあります。

  • show()
  • hide()
  • toggle()
  • slideUp()
  • slideDown()
  • slideToggle()
  • fadeOut()
  • fadeIn()
  • fadeTo()
  • animate()

エフェクト：基本的な使い方

jQuery(function($) {
  $('div p').slideUp(500, function() {
    $(this).slideDown(500);
  });
});

clickしたらslideUp＆slideDownを実行するようにしてください！

jQuery(function($) {
  $('div p').each(function() {
    $(this).click(function() {
      $(this).slideUp(500, function() {
        $(this).slideDown(500);
      });
    }):
  });
});

Ajax (XMLHttpRequest)

これが無いとWebアプリ作れないしー。

jQuery.ajax()を使うと細い設定が行えますが、今回はさくっと組み込むのに便利なエイリアスを使います。

  • jQuery.get()
  • jQuery.post()
  • jQuery.getJSON()
  • $().load()

Ajax：使い方

jQuery(function($) {
  $.get('example.html', {hoge: 'fuga'}, function(html) {
    // example.html?hoge=fuga
    $('#target').html(html);
  });
});

パラメータは省略することも可能

jQuery(function($) {
  $.get('example.html', function(html) {
    $('#target').html(html);
  });
});

post()やgetJSON()も同じように使えます。

Ajax：使い方

jQuery(function($) {
  $('#target').load('example.html', {hoge: fuga}, function() {
    // 他に処理があれば書く
  });
});
